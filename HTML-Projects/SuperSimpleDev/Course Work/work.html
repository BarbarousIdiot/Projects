GNU stands for GNU's Not Unix was founded by Richard Stallman in the 1980s to create a complete free and opensource operating system which was going to be an alternative to UNIX.

Linux Kernel was created by Linus Torvalds in 1992 while he was still in College. He created this becuase he wasn't having a good time using Mimix(another Unix Based Operating System) . So he decided to create his own and opted GNU for the operating system. Linux Kernel is based on UNIX. In GNU/Linux based operating Systems , the GNU is the OS and the kernel is Linux.
